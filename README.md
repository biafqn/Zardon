## 📝 Sobre o Projeto

Este projeto nasceu de uma necessidade corporativa para substituir métodos arcaicos (como livros de ocorrência em papel) por uma solução digital. O **Zardon** permite que a equipe de segurança, recepção ou funcionários registre itens rapidamente, gerando um banco de dados confiável para consultas futuras.

---

## 💼 Contexto da Demanda

O projeto foi estruturado para atender à necessidade de um estabelecimento. O foco principal foi a usabilidade, permitindo que funcionários de qualquer setor consigam registrar um item perdido em poucos segundos.

## ✨ Funcionalidades

- **📥 Cadastro de Itens:** Registro com foto (opcional), descrição, local, data e quem o encontrou.
- **📑 Gestão de Status:** Controle se o item está "Em Estoque", "Devolvido" ou "Encaminhado para Doação/Descarte".
- **🔍 Filtro de Busca:** Localização rápida por nome do objeto ou período de tempo.
- **👤 Comprovação de Retirada:** Campo para registro de dados do proprietário no ato da devolução.
- **📊 Relatórios:** Visão geral de quantos itens entram e saem mensalmente.
